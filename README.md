# JobMagitha Portal

A responsive job portal built with HTML, CSS, and TypeScript.

## Features

- Search jobs by title, company, location, or role
- Filter by company and role
- Save and remove bookmarked jobs
- Responsive card-based layout with hover interactions

## Run

Open `index.html` in a browser.

## TypeScript

Source code lives in `src/app.ts`.

The project also includes a browser-ready compiled file at `dist/app.js` so it works immediately in environments where `tsc` is not installed.
